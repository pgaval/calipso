Installation Prerequisites and Supported Platforms

Calipso leverages an open, industry standard technology stack allowing it to run on a broad variety of platforms. For this specific installer you will need:

  • File system rights for the directory you would like Calipso and supporting files to be installed.
  • Either an Administrative account for a running MySQL database server (v5.1+) or details of an empty database and a corresponding user with read/write permissions on it. Using an Administrative account allows you to create a database and the corresponding user if needed
  • File system rights for the deploy directory of a Servlet container running on Java 1.6 or later.
